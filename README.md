# Assignments-base-on-Bioinformatics

# 🧬 Bioinformatics & Computational Biology Projects

## Overview

This repository showcases a collection of bioinformatics and computational biology projects completed as part of graduate-level coursework at the University of Cincinnati. These projects demonstrate hands-on experience in sequence analysis, RNA-seq data processing, dynamic programming algorithms, and deep learning applications in biological data.

The work integrates concepts from machine learning, algorithm design, and biological data analysis to solve real-world problems in genomics and proteomics.

---

## 🔬 Projects Included

### 1. Read Mapping using Bowtie

* Implemented sequence alignment workflows to map short DNA reads to a reference genome using Bowtie
* Analyzed alignment accuracy and explored genome mapping techniques
* Gained practical experience in handling high-throughput sequencing data

---

### 2. 🧬 Open Reading Frame (ORF) Identification

* Developed methods to identify open reading frames in mitochondrial DNA sequences
* Extracted coding regions and analyzed potential protein-coding sequences
* Applied biological sequence parsing and pattern recognition techniques

---

### 3. 🧬 RNA-Seq Analysis using Kallisto

* Performed transcript quantification using Kallisto on RNA-seq datasets
* Computed gene expression levels using TPM normalization
* Compared transcriptome references (RefSeq vs Ensembl) to analyze expression variability

---

### 4. 🧩 Sequence Alignment using Dynamic Programming

* Implemented Needleman-Wunsch (global alignment) and Smith-Waterman (local alignment) algorithms
* Generated optimal sequence alignments using dynamic programming techniques
* Evaluated alignment quality using scoring matrices and gap penalties

---

### 5. 🧬 Protein Sequence Embedding using ESM2

* Fine-tuned transformer-based protein language model (ESM2) for sequence representation
* Generated protein embeddings for downstream biological analysis
* Explored deep learning applications in protein structure and function prediction
